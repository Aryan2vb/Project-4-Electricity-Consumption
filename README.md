# Project 4: Mining Electricity Consumption Data for Resource Allocation and Supply Chain Enhancement

## Problem Statement

Electricity demand is not uniform, it varies across regions, industries, and time (daily, weekly, seasonal, and yearly cycles). Efficient allocation of resources requires identifying where the demand is rising, where supply bottlenecks exist, and what areas (geographic or industrial) need immediate investment in grid infrastructure, generation, or renewable energy support. This project challenges students to analyze electricity consumption data (at household, commercial, industrial, and/or regional levels) to propose data-driven insights on where governments and utility companies should invest in infrastructure, optimize distribution, or enhance supply chains.

## Dataset

[U.S. Energy Information Administration (EIA) Open Data](https://www.eia.gov/opendata/browser/electricity)

[Data Api Docs](https://www.eia.gov/opendata/documentation.php)
